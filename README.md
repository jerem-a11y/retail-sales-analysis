# Retail Sales Data Analysis

## Project Overview

This project analyzes retail sales data using Python to uncover insights about customer purchasing behavior, product performance, and revenue distribution.

The analysis focuses on:

- Total sales by gender
- Price vs total revenue relationship
- Product sales performance
- Regional price distribution
- Monthly sales trends

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset

The dataset contains the following fields:

- Gender
- Product
- Quantity
- Price
- Region
- PurchaseDate

A new feature **Total** was created using:

Total = Quantity × Price

## Key Analysis

### 1. Sales by Gender
Identifies which customer segment contributes the most revenue.

### 2. Price vs Total Sales
Examines how product pricing affects total revenue.

### 3. Product Sales Aggregation
Analyzes the total revenue generated per product.

### 4. Regional Price Distribution
Visualizes price contributions by region.

### 5. Monthly Sales Trend
Extracts month from purchase dates to analyze seasonal trends.

## Example Visualizations

- Bar Chart: Total Sales by Gender
- Line Plot: Price vs Total
- Pie Chart: Regional Price Distribution
- Histogram: Data distribution
